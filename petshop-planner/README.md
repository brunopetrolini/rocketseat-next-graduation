# 🐾 Mundo Pet

O **Mundo Pet** é uma plataforma ágil e intuitiva desenvolvida para facilitar a gestão de agendamentos em pet shops. Com uma interface limpa e responsiva, o sistema permite que a equipe do pet shop controle a agenda do dia de forma eficiente, visualizando rapidamente os horários ocupados e adicionando novos atendimentos em poucos cliques.

## ✨ Funcionalidades Principais

- **Visão Geral da Agenda:** Visualize todos os clientes e serviços agendados para o dia, organizados por períodos (Manhã, Tarde e Noite).
- **Filtro de Data:** Navegue facilmente por dias anteriores ou futuros através de um seletor de calendário.
- **Novo Agendamento Rápido:** Adicione um novo atendimento informando o nome do tutor, nome do pet, telefone de contato, a descrição do serviço desejado (ex: banho, tosa, vacinação), data e horário.
- **Cancelamentos:** Remova agendamentos da lista de forma simples caso haja desistência do cliente.
- **Design Moderno e Responsivo:** Interface pensada na usabilidade (UI/UX) com suporte nativo a temas escuros (Dark Mode), facilitando a visualização.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído com ferramentas modernas do ecossistema JavaScript, visando performance e facilidade de manutenção:

- **[Next.js](https://nextjs.org/)** (React Framework)
- **[TypeScript](https://www.typescriptlang.org/)** (Tipagem Estática)
- **[Tailwind CSS](https://tailwindcss.com/)** (Estilização e Layout)
- **[Biome](https://biomejs.dev/)** (Linter e Formatação de Código)

## 🛠️ Como rodar o projeto localmente

Para rodar o Mundo Pet na sua máquina, você precisa ter o [Node.js](https://nodejs.org/) instalado.

**1. Clone o repositório e instale as dependências:**
```bash
# Instale as dependências usando npm (ou yarn/pnpm se preferir)
npm install
```

**2. Inicie o servidor de desenvolvimento:**
```bash
npm run dev
```

**3. Acesse a aplicação:**
Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o sistema em funcionamento.

---

> **Dica de Desenvolvimento:** O projeto utiliza o Biome para manter a consistência do código. Você pode rodar `npm run lint` para checar problemas de formatação ou `npm run format` para corrigi-los automaticamente.
